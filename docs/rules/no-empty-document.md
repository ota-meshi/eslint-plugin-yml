# yml/no-empty-document

> disallow empty document

- :gear: This rule is included in `"plugin:yml/recommended"` and `"plugin:yml/standard"`.

## :book: Rule Details

This rule reports empty mapping documents.

<eslint-code-block>

<!-- eslint-skip -->

```yaml
# eslint yml/no-empty-document: 'error'
---
# ✓ GOOD
"GOOD": "foo"
...
---
# ✗ BAD
...
```

</eslint-code-block>

Nothing.
