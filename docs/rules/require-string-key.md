# yml/require-string-key

> disallow mapping keys other than strings

## :book: Rule Details

This rule reports mapping keys defined with an other than a string.

<eslint-code-block>

```yaml
# eslint yml/require-string-key: 'error'

# ✓ GOOD
"GOOD": 42

# ✗ BAD
42 : "BAD"
```

</eslint-code-block>

## :wrench: Options

Nothing.
