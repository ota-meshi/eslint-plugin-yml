# yml/no-empty-key

> disallow empty mapping keys

- :gear: This rule is included in `"plugin:yml/recommended"` and `"plugin:yml/standard"`.

## :book: Rule Details

This rule reports empty mapping keys.

<eslint-code-block>

<!-- eslint-skip -->

```yaml
# eslint yml/no-empty-key: 'error'

# ✓ GOOD
"GOOD": "foo"

# ✗ BAD
: "BAD"
```

</eslint-code-block>

Nothing.
